# TodoApp

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 16.0.4.
It is a simple yet practical Todo application built as part of Fernando Herrera's Redux course. The main goal of the app is to demonstrate how to manage global state using Redux in a clean and scalable way.

The application allows users to create, toggle, and delete tasks, as well as filter them based on their completion status (all, completed, or pending). It focuses on core Redux concepts such as actions, reducers, and the store, while also showing how to connect Redux with a React user interface.
